What is the use of the open graph protocol?

  • The Open Graph protocol enables any web page to become a rich object in a social graph.

Where are open graph tags inserted in an HTML document?

  • Open graph tags are in tags in the of a document.

What prefix is used for Twitter's version of Open Graph tags?

  • The prefix that's used for Twitter's version of Open Graph tags is "twitter:"
